# WhisperAnnouncer

![Addon Logo](https://raw.githubusercontent.com/MaQuiNa1995/WhisperAnnouncer/master/Images/AddonLogo.png)

## What is WhisperAnnouncer?

**WhisperAnnouncer** is a World of Warcraft addon that displays an on-screen notification whenever you receive a whisper message — either a standard in-game whisper or a Battle.net whisper.

The notification clearly shows the name of the player who sent the message, allowing you to instantly know who is contacting you without checking the chat window.

---

## Features

- On-screen notification when receiving a whisper.
- Displays the sender’s name clearly.
- Supports:
  - Regular in-game whispers
  - Battle.net whispers
- Lightweight and simple.
- No configuration required.
